# 关卡花色统计

本目录是一个本地网页工具：

- 上传关卡视频
- 点击“输出结果”
- 输出总花色组数，以及每一种花色名称和数量
- 可配置视觉命名 Key，新花色写入消除记录前会根据缩略图直接命名为“粉色树莓”这类真实名称
- 可在“消除记录”里修改单条花色或删除误识别项，汇总会自动刷新

当前内置了 `10.mp4` 里出现过的 10 个花色模板作为识别参考，并从美术规划表生成了候选词库。命名时优先用模板真实名；遇到新图案时会在记录消除前调用视觉模型，结合缩略图和词库直接生成真实名称，避免“颜色花色一”或“待识别”这类占位代称。

视觉命名默认请求 `https://api.openai.com/v1/responses`。如果使用 OpenAI Key，请填写 OpenAI 平台的 `sk-...` Key；如果使用 Lumos 或其它平台 Key，需要同时把“API 地址”改成该平台提供的 OpenAI 兼容地址。API 地址可以填以 `/v1` 结尾的 base，也可以直接填完整的 `/responses` 或 `/chat/completions` 地址；如果平台不支持 Responses API，页面会自动再试一次 Chat Completions。

## 本地打开

需要视觉命名时请用本目录的本地服务启动，页面会通过本地代理请求模型接口，避免浏览器跨域导致 `Failed to fetch`：

```powershell
python server.py --port 8778
```

然后访问 `http://127.0.0.1:8778/`。如果只是做不带视觉命名的视频统计，也可以直接打开 `index.html`。
